# 📚 Flashcard Quiz App

A simple and interactive Flashcard Quiz App developed using **Python** and **CustomTkinter** as part of the **CodeAlpha App Development Internship**.

## 🚀 Features

- 📖 View flashcards
- 👀 Show answers
- ⬅️ Previous and Next navigation
- ➕ Add new flashcards
- ✏️ Edit flashcards
- 🗑️ Delete flashcards
- 💾 Flashcards are saved using JSON

## 🛠️ Technologies Used

- Python
- CustomTkinter
- JSON

## ▶️ How to Run

1. Install Python.
2. Install CustomTkinter:

```bash
pip install customtkinter
```

3. Run the application:

```bash
python main.py
```
